## Getting Started

1. Clone this repository in your local machine
2. Wait for dependencies to be installed
3. Open, appsettings.Development.json and change the connection string to your desired SQL Server Database Connection. You can left the default one if you want. 
4. Run 'dotnet ef database update' to apply migrations in the database.
5. Run the application
